Vanilla HTML demo

One <link> tag. Plain BEM classes. No JavaScript, no build step, no dependencies.

This page is built with nothing but a stylesheet from unpkg and standard HTML. Drop klods-css into any Rails app, WordPress theme, or plain HTML file and start building immediately.

Add the stylesheet to your <head>:

<link rel="stylesheet"
  href="https://unpkg.com/klods-css/dist/klods.min.css">

Then use the classes shown below.

Buttons

<button class="klods-button klods-button--primary" type="button">
  Save
</button>

Alerts

Info — something worth knowing.
Success — changes saved.
Warning — double-check before continuing.
Danger — this action is irreversible. 
<div class="klods-alert klods-alert--success">
  Changes saved.
</div>

Cards

Standard

Surface colour, border, rounded corners.

Elevated

Drop shadow instead of border.

</div>

Layout

This page itself demonstrates the full shell. The outer wrapper carries three classes: 

<div class="klods-page
           klods-page--with-sidebar
           klods-page--sticky-header">

  <header class="klods-header">…</header>
  <aside  class="klods-sidebar">…</aside>
  <main   class="klods-content">…</main>
  <footer class="klods-footer">…</footer>

</div>

Add klods-content--narrow to cap the content width at a comfortable reading measure. Add klods-content--wide to stretch it to full page width.
